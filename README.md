Repository ini ditujukan untuk memenuhi Tugas Besar Pemrograman Berorientasi Objek (IF2010) tahun ajaran 2025/2026 


# Cara Menjalankan Program:
1. pastikan sudah menginstall extention GUI 
2. running program di app/main
3. run manual ketik berikut di terminal : 
java -cp bin app.Main 
4. Atau bisa juga dengan menjalankan tombol run java langung pada kelas Main.java
